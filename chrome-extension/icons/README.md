# Icons

Generate PNG icons from the SVG:

```bash
# Using ImageMagick
convert icon16.svg icon16.png
convert -resize 48x48 icon16.svg icon48.png
convert -resize 128x128 icon16.svg icon128.png

# Or use any image editor to create:
# - icon16.png (16x16)
# - icon48.png (48x48)
# - icon128.png (128x128)
```

For now, you can use any 16x16, 48x48, and 128x128 PNG files as placeholders.
